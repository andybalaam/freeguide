Run the nightly build of FreeGuide

Note: Before you do this you should take a backup of your FreeGuide install directory and your working directory and registry settings and keep them somewhere safe. You will be running bleeding-edge code, so there may well be bugs.
Be prepared to lose your favourites, settings etc. If it all goes badly wrong, uninstall FreeGuide, delete the freeguide-tv node in the registry (or rm -rf ~/.java/.userPrefs/org/freeguide-tv on Linux) and install the latest version of FreeGuide, and it should all be back to normal.

You can get a build of FreeGuide from the very latest SVN version here:

http://users.telenet.be/richard.turnbull/Free Guide Last Version/

The Windows "-with-xmltv" installer also includes the very latest version of XMLTV.

The installers can be installed exactly the same way you would install a normal FreeGuide release.

Also in this directory is the file FreeGuideUpdate.rar, which just includes the JAR files for FreeGuide - you can unrar this file and copy them over the current FreeGuide JARs to get the latest version without using an installer.

There is more information on Richard Turnbull's FreeGuide site.
