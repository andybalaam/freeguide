"Choose Channels" is not working - what should I do?

If your grabber fails to configure or run because it can't seem to get any data from the Internet, it may be because you need to use a proxy server.

Instructions on how to set up FreeGuide to use a proxy server are here: Configure Proxy On Windows or Configure Proxy On Linux.
