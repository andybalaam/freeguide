Installing FreeGuide on other operating systems

We'd love to put instructions up on how to install FreeGuide on other operating systems.

You will need OpenJDK/IcedTea or Sun's Java and XMLTV, and, theoretically, it should work.

Please Contact Us if you manage it, or if you need help trying!
