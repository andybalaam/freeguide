FreeGuide Bug Report

You can use this page to report any bugs or problems you have with FreeGuide. It is intended to help you provide as much useful information as possible when you report a bug. When you click "Submit" at the end of this form, and email will be sent to the FreeGuide developers' mailing list on freeguide-tv-devel@lists.sourceforge.net. If you prefer, you may email this list directly, but try to include as much information as possible.

Operating system E.g. Windows XP, Red Hat Linux 9.
FreeGuide version Look in the title bar of FreeGuide to find out what version you are using.
Java version Open a DOS, Cmd or terminal window and type "java -version" to find out what version of Java you are using. Copy the output into this box.
Show details information If you're having a problem with downloading listings, and you get an "Execution Error" message, click on "Show details" in the downloading window and copy and paste the information in the window that pops up into this box.
Command window output If you can't get any information about what's going wrong, try running FreeGuide from within a DOS, Cmd or terminal window and copy and paste everything that appears in the terminal into this box. On Windows, open a DOS or Cmd window and type cd "C:\Program Files\FreeGuide" then press RETURN and type java -jar FreeGuide.jar to run FreeGuide. There are some instructions on how to copy text out of a dos box available. When you've copied the output, paste it into the box to the left by clicking in the box and pressing Ctrl-V.
