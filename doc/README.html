Introduction

FreeGuide is a TV guide program. It uses parser programs to extract TV information from web pages and stores them for viewing without the need to connect to the Internet. The viewer allows the user to view television listings and create customised TV guides by selecting programmes and by building up a favourites list.

It works with listings for Canada, Denmark, Finland, Germany, Hungary, Italy, Japan, the Netherlands, New Zealand (Linux only), Spain, the UK and the US.

Installing FreeGuide

If you are in North America, you need to get a Data Direct account from Zap2it before you install FreeGuide. To do this, go to http://labs.zap2it.com and click the "New User? Sign-Up" link. Use the certificate code TUYO-DCXM-CBUT. Make a note of the username and password you choose as you will need to enter them during the installation process.

Windows

  • Make sure you've got Java version 1.4 or above. (How do I know?)
  • Set your timezone. (How?)
  • Download the file FreeGuide-X.X-Win-Install.jar from the download page and double-click on it to run it, and follow the instructions to install FreeGuide and configure your grabber.
  • Find the file "C:\Program Files\freeguide-tv\FreeGuide.jar" in "My Computer" and double-click on it to run FreeGuide. If you like, you can make a shortcut to this file (by right-clicking and choosing "Create Shortcut") and put it on your desktop (by dragging it onto the desktop).

If you have problems try looking for an answer in the FAQ.
