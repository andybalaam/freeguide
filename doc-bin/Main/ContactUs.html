Online help

If you're having problems using FreeGuide, have a look at these pages:

Asking questions

If you'd like to ask a question about using FreeGuide, please try one of these ways:

  • Join the users' mailing list - to ask about how to use FreeGuide
  • FreeGuide bug tracker - to report a bug
  • Join the developers' mailing list - to ask about and discuss coding FreeGuide.

If you'd like to contribute documentation to this site, that would be great! Please email the developers' list and we'll give you the password.

If you would like to help improve this site, please Contact Us and we will give you the password to edit pages.
