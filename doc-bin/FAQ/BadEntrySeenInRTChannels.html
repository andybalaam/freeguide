If you see this kind of output when you try to download listings in the UK: 

Running command: tv_grab_uk_rt --quiet --config-file "/home/andy/.freeguide/xmltv-configs/uk_rt.conf"
Bad channel entry seen in RT channels.dat: 	 at /usr/bin/tv_grab_uk_rt line 490.
Error executing grabber: Content is not allowed in prolog.
Result code = 9

You need to get a later version of XMLTV. If you upgrade to 0.5.54 or later it should work. There are instructions here: Upgrading XMLTV.

This may be related to this XMLTV bug: XMLTV Bug 1880681.
