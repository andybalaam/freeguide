Résumé

Quand FreeGuide démarre, il peut vérifier sur le site web si une nouvelle version est disponible. Il nous est utile de savoir combien de personnes se servent de FreeGuide, une trace est conservée des interrogations faites chaque jour.

Vous pouvez décider de désactiver cette vérification complètement (choisissez "Non"), ou vous pouvez activer la vérification et choisir quelles informations vous concernant seront conservées.

L'information la plus utile pour nous est lorsque vous choisissez un pseudo qui vous identifie de manière unique. Cela veut dire que nous pouvons connaître le nombre de personnes distincte qui utilisent FreeGuide. Sinon, vous pouvez aussi autoriser votre IP à être stockée (ce qui arrive dès vous visitez un site web). Ce la nous permettra déjà de nous faire une idée du nombre d'utilisateur, mais cela sera moins précis car certains fournisseurs d'accès internet vous donne une IP différente à chaque connexion. Enfin, vous pouvez décider de faire une vérification mais de nous ne donner aucune info, oui vous le pouvez. Une connexion sera malgré tout enregistrée mais aucune information vous concernant ne sera enregistrée.

Si cela suscite une quelconque interrogation, n'hésitez pas à envoyer un mail à freeguide-tv-devel@lists.sourceforge.net.

A quoi servent ces informations ?

A l'heure actuelle ces informations ne sont pas utilisées pour quoi que ce soit. Dans le futur, elles seront peut-être utilisées dans les buts suivants 

  • Pour satisfaire l'ego des développeurs de FreeGuide.

  • Pour convaincre une boite de nous donner tout plein d'argent car nous avons plein d'utilisateurs.

En aucune cas ces données ne seront utilisées pour identifier des individus. Elles ne seront jamais vendues ou données à quiconque voudrait les utiliser à des fins commerciales. Elles ne seront jamais utilisées pour vous envoyer de la pub, du SPAM, etc.

En gros FreeGuide est fait par passion. Un jour il y aura peut-être un peu d'argent généré par ce projet pour moi (Andy Balaam), et cela pourrait être un moyen pour convaincre certaines personnes qu'elles devraient payer pour être le fournisseur de contenu pour un pays donné car ils auront plein de consultations de leur site grâce à cela. Pour faire ça, ils voudront sans doute savoir qu'il y plein d'utilisateurs de FreeGuide, et c'est un moyen de le prouver.

Quels sont les détails techniques ?

Quand FreeGuide démarre, si vous avez choisis de le faire, FreeGuide vérifie cette URL : http://freeguide-tv.sourceforge.net/VERSION.php. Le code de cette page est disponible dans src/program/lib/fgspecific/VersionCheckerThread.java dans les fichiers sources. En fonctions de vos réglages d'intimité, quelques valeurs sont ajoutées à la requête GET, fournissant soit votre pseudo, ou une fausse IP (0.0.0.0) si vous décidez de ne pas fournir d'information. Si vous choisissez de fournir votre IP, l'URL est laissée telle quelle et votre IP est détectée automatiquement. La version de FreeGuide que vous utilisez est ensuite ajoutée à cette URL.

La date, l'identifiant (pseudo, IP ou fausse IP) et la version sont stockés dans un fichier situé sur le serveur. Le script PHP qui remplis ce fichier est disponible dans doc/VERSION.php.
