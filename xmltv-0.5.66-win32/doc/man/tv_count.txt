NAME
    tv_count - Count (and print) the number of channels and programmes in an
    XMLTV file.

SYNOPSIS
    tv_count FILE

DESCRIPTION
    Read XMLTV listings and print a count of the number of channels and
    number of programmes in the file.

EXAMPLE
    Use "tv_count listings.xml" to count programmes in the file called
    listings.xml.

SEE ALSO
    xmltv(5)
