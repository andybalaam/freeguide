FreeGuide Windows README

(contributed by Sam Pye with minor modifications by Andy Balaam)

At the moment this readme is only tested with FreeGuide 0.3 on Windows XP with XMLTV 0.4.5, grabbing listings for North America. Please send feedback on how you get on with other versions. When XMLTV 0.5 comes out, some of the problems addressed by this README should be fixed, so you may be able to follow the standard README instead.

Installation Instructions

  1. Download and install Java (Java Runtime Environment, Version 1.4 at least)
  2. Download and extract the XMLTV tools (version 0.4.5) to c:\xmltv
  3. Set your environment variable "TZ" to your time zone. ("EST5EDT" refers to New York timezone, so you'll need to adjust it if this kind of thing worries you.)

    4. With Windows 2000/XP:

    • Right Click "My Computer" --> Properties
    • Choose the Advanced Tab and click the Environment Variables button at the bottom middle of the window.
    • Click the "New" button in the Global Variables section.
    • Variable Name: TZ
    • Variable Value: EST5EDT
    • Click OK
    • Click OK

    With Windows 95/98/ME:

    • Add a line "set TZ=EST5EDT" to the file c:\autoexec.bat

  4. Download and extract FreeGuide 0.3 to c:\xmltv
  5. Open a command window (on Win XP: Start --> Run --> CMD [enter]) (on Win 98: Start --> Programs --> Accessories --> MS-DOS Prompt)
  6. Change to c:\xmltv directory (cd c:\xmltv)
  7. Type tv_grab_na --configure [enter] (this will configure the XMLTV tools, signal provider + channels)
    1. A file by the name of .xmltv_na is created as the configuration file, this file must be in the same location as the tv_grab_na executable.
  8. Type tv_grab_na --output c:\xmltv\.xmltv\freeguide-tv\unprocessed_listings.xml [enter] (this will download our channel listings)
    1. Downloading can take 20 mins if your updating ~40 channels - it is fetching 7 days of listings
  9. Type tv_split --output c:\xmltv\.xmltv\freeguide-tv\%%channel-%Y%m%d.fgd c:\xmltv\.xmltv\freeguide-tv\unprocessed_listings.xml
    1. Like usual parsing text takes a little time, be patient, if you see a few "unexpected" error messages you can ignore them
    2. Splitting is almost finished when you see this text: "Splitting listings by date and channel."
  10. Now check that the files unprocessed_listings.xml and lots of ".fgd" files are in the directory C:\xmltv\.xmltv\freeguide-tv - if not, something has gone wrong.

    11. *** At this point we should be able to launch FreeGuide to alter our options and then view our downloaded listings. When using XMLTV 0.4.5 we need to launch FreeGuide in a slightly strange way to allow tv_grab_na to find the config file.

  11. Open a CMD prompt, reference Step 4 if you forget how, and change to c:\xmltv
  12. Type java -classpath c:\xmltv\freeguide-tv\gui-java\FreeGuide.jar FreeGuide

    13. FreeGuide will take you through an initial configuration wizard. Answer the questions and click "Finish". You'll get an error, so got to the Options screen as it recommends, and set up the options as follows: (To get to the Options screen if you're not already there, click Tools then Options.)

  13. Go to the misc tab and make sure the settings are exactly as follows:
    WorkingDirectory: c:\xmltv\.xmltv\freeguide-tv\
    Stylesheet file: c:\xmltv\.xmltv\freeguide-tv\guide.css
    XMLTV Directory: C:\xmltv\
    Grabber Config File: c:\xmltv\.xmltv_na
    Day Start Time: 06:00

    14. When you click OK you should see the listings for your chosen channels for today. Now you can go back to the Options screen and change your selected channels on the Channels tab, and see the main README for how to use FreeGuide.

    When you run out of listings you'll need to run tv_grab_na and tv_split again from the CMD line, as FreeGuide's built-in downloader doesn't work properly in Windows yet.

    If you have any problems, contact us and we'll sort you out.
